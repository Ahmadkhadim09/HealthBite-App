The origin is set to https://github.com/account — that's a placeholder. You need to update it to your actual GitHub repo URL. Here's how:

Step 1 — Create a new repo on GitHub
Go to github.com/new → name it health-bite → click Create repository (leave everything else blank)

Step 2 — Copy your repo URL
It'll look like: https://github.com/YOUR-USERNAME/health-bite.git

Step 3 — Run this in the Shell (replaces the old placeholder):

git remote set-url origin https://github.com/YOUR-USERNAME/health-bite.git
git push -u origin main

Step 4 — When it asks for password
Don't use your GitHub password — use a Personal Access Token:

GitHub → click your profile photo → Settings
Scroll down → Developer settings → Personal access tokens → Tokens (classic)
Generate new token → tick repo → Generate → copy the token
Paste it as the password in the Shell
